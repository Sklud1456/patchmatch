# Patchmatch flask demo

## 本地预览

### 源码

>依此按照下面命令

```
D:\Github\patchmatch
$ npm install
D:\Github\patchmatch
$ virtualenv venv

//进入venv\Scripts目录
D:\Github\patchmatch\venv\Scripts
$ activate
D:\Github\patchmatch
(venv) $ pip install -r requirements.txt
D:\Github\patchmatch
(venv) $ python manage.py

//开启另一个终端(或者像上一个方式直接打开/dist/index.html)
D:\Github\patchmatch
$ npm run dev
```
>等待浏览器打开 http://localhost:8080
